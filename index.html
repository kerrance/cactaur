Moogle.js

Hello!

Thank you for checking out this simple JavaScript pop-up modal. I want to write something very simple and clean to test my skills.

There are a few ways to enable the pop-up modal:

The most basic method is to activate by clicking or tapping on the 'Kupo!' button below.

The second method is to simply scroll the page. When you scroll over half way through the page, the pop-up modal will appear.

The code for this project is freely available on GitHub, so please feel free to make use of it if you would like to use something similar on your own website. All I ask is that you 'Star' the repository for future reference, or credit the repository in your README file.
